# Toward-Reliable-Early-Glaucoma-Detection-A-Quality-Aware-EfficientNet-Approach

Glaucoma is a chronic ocular disease that can lead to irreversible blindness if not detected at an early stage. However, the diagnostic process continues to face significant challenges due to image quality variations and the lack of model transparency. This study proposes an automated binary classification approach using the EfficientNet-B0 architecture, enchanted with a quality-weighted inference mechanism to improve clinical reliability. The model was developed using a specialized glaucoma dataset comprising 747 fundus images from 288 patients, utilizing a patient-level stratified split to ensure evaluation validity. To address quality fluctuations, the system automatically reduces predictive certainty or recommends image re-acquation for low-quality inputs to mitigate misdiagnosis risks. Experimental results demonstrate strong performance, achieving an accuracy of 98%, sensitivity of 99% for glaucomatous cases, and an area under the curve of 0.99. Gradient-weighted class activation mapping visualizations further confirm that the model consistently prioritizes clinically relevant features, specifically the optic disc and neuroretinal rim.
